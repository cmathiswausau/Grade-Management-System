# Grade Management System

A C# application built using .NET that manages student grades, assignments, and performance calculations using object-oriented design principles.

## 📌 Overview

This project was developed using instructor-provided requirements and structure to apply object-oriented programming concepts in a structured academic scenario. The implementation focuses on organizing student data and performing calculations such as averages and overall performance.

## 🎯 Key Features

* Management of student records and grade data
* Calculation of averages and overall performance
* Use of collections to store and process data
* Structured class relationships for maintainability

## 🧠 Concepts Demonstrated

### Object-Oriented Programming

* Class design and relationships
* Encapsulation of data and behavior
* Use of properties and methods

### Data Handling

* Working with collections (e.g., lists of students and grades)
* Performing calculations such as totals and averages
* Managing structured data efficiently

### Code Execution & Debugging

* Step-through debugging to trace execution flow
* Evaluation of expressions and method calls at runtime
* Identifying and resolving logical errors

### Development Practices

* Modular code organization across multiple files
* Readable and maintainable code structure
* Implementation based on defined requirements
